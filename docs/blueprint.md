# **App Name**: Presell Pro

## Core Features:

- Presell Page Editor: A complete editor to customize the landing page, and add images and layouts to it.
- HTML Code Generation: Generates an HTML code based on the configuration, to be deployed on any server.
- Preview: Displays a real time desktop and mobile preview.
- Automatic adjustments: Automatically suggests best layout for each landing page.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5) to convey trust and professionalism.
- Background color: Very light gray (#F9FAFB), creating a clean, modern backdrop.
- Accent color: Teal (#34D399) to highlight interactive elements and calls to action, providing a sense of vibrancy.
- Headline font: 'Space Grotesk', sans-serif, for a techy, modern feel. Body font: 'Inter', sans-serif, for clear and readable content.
- Use clean, line-based icons from a set like Lucide to maintain a consistent and modern aesthetic.
- Employ a grid-based layout with ample white space to ensure a clean and organized presentation.
- Use subtle animations for transitions and interactions to enhance the user experience without being distracting.
